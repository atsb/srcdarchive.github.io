SRCD Demo Archive

What is SRCD

SRCD is the continuation of Compet-N and Competition Doom since the current maintainer of the website for these has stated that by July 2022, the website will be removed. This ensures the years of demos and the legacy these have with the Doom Community will continue to be available

How to contribute new Compet-N demos

  1. DOOM.EXE/DOOM2.EXE can only be used (via DOSBox)
  2. Create an issue, with the Naming Conventions and Naming Examples forms filled out and with an issue title of "NEW ENTRY - "
  3. Include the forms, the demo LMP file and any other information as an attachment to the issue (player info etc..) inside a zip file. Please see 'compet-n/admin/examples'

Verification will then be performed and once verified, will be added to the archive.

What source port can we use for Competition Doom (CNDoom)?

The only source port that can be used is SRCD-Doom, which is 'the' source port for strict vanilla competition demos (for Windows and macOS)

SRCD-Doom

How to contribute new SRCD Demos

This is largely the same as for Compet-N however the rules are much more relaxed. Please see any existing srcd demo zip file for the content that you are expected to provide.

  1. SRCD-Doom (windows or mac) must be used
  2. Create an issue with the title "NEW SRCD ENTRY - "
  3. Place a zip file with your demo and any text information you would like to add as an attachment to the issue (see the above links for Compet-N) for examples
  4. zip file and demo name should be (example): doom1-speed-e3m7-0005289.zip

Are Tyson runs allowed in SRCD

Yes, any run that is allowed in Compet-N is allowed in SRCD
